## Übersicht

Die Meta Quest ist die einfachste Art, VR zu erleben — einfach aufsetzen, spielen. Keine Kabel, keine externe Hardware nötig. Für anspruchsvollere PCVR-Titel kommt Virtual Desktop zum Einsatz.

## Virtual Desktop — PCVR übers Netz

Virtual Desktop streamt den PC-Desktop (Windows Server im Heimnetz) drahtlos ins Headset. Mit WiFi 6 und aktiviertem Game-Streaming-Modus ist die Latenz kaum spürbar.

```
Meta Quest ──WiFi 6──→ Router ──LAN──→ Windows Server
               Latenz ~10-15ms (5GHz, kurze Distanz)
```

## Plex VR

Über die Plex-App auf der Quest können Filme und Serien vom QNAP NAS in einem virtuellen Kino angeschaut werden — ideal für lange Flüge oder wenn der TV belegt ist.

## Play Space

Das Guardian-System merkt sich die Raumgrenzen im Wohnzimmer. Einmal eingerichtet, erscheint das virtuelle Gitter automatisch sobald man sich einer Wand nähert.

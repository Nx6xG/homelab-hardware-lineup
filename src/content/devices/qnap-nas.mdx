## Übersicht

Die QNAP NAS ist das zentrale Nervensystem für alle Daten im Homelab. Alle anderen Geräte — Raspberry Pis, Windows Server, MacBook — sichern ihre Daten hierher.

## Share-Struktur

| Share | Zweck |
|-------|-------|
| `media` | Plex-Bibliothek (Filme, Serien, Musik) |
| `backups` | Automatische Backups aller Geräte |
| `dokumente` | Persönliche Dokumente |
| `photos` | Foto-Archiv |
| `timemachine` | macOS Time Machine für MacBook |

## Backup-Eingang

Die NAS ist das Ziel für Backups von:
- **Home Assistant**: Tägliche HA-Snapshots
- **Raspberry Pi 5**: Portainer-Volume-Backups (via Script)
- **Windows Server**: Game-Server-Worlds via Robocopy
- **MacBook Pro**: Time Machine über SMB

## Plex-Setup

Plex läuft nativ auf der NAS. Für lokale Clients (Apple TV, iPhone, iPad) funktioniert **Direct Play** ohne Transcoding. Für unterwegs wird auf dem MacBook oder direkt über die Plex-App gestreamt.

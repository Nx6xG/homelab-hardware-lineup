## Übersicht

Der Windows Server ist dediziert fürs Gaming. Das Ziel: Freunde können ihre Game-Server selbst verwalten, ohne SSH oder Terminal-Kenntnisse.

## AMP — Application Management Panel

AMP stellt für jede Game-Server-Instanz eine eigene Web-UI bereit. Start, Stop, Konsole, Backup, Mod-Updates — alles per Browser.

## Backup-Strategie

Jede Nacht um 03:00 Uhr kopiert ein Robocopy-Script alle World-Saves auf den QNAP NAS:

```powershell
Robocopy "D:\AMP\Instances\*\Saves" "\\qnap\backups\gameservers" /MIR /R:3 /W:10
```

30 Tage Retention, wöchentliche Snapshots für 90 Tage.
